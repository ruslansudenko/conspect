    Конкатенация(concatination - сцепление, соединение) - это операция объединения или склеивания строк или последовательностей символов в одну строку.
    example: 
    const str1 = "Привет, ";
    const str2 = "мир!";
    const result = str1 + str2;
    console.log(result); // Вывод: "Привет, мир!"

         function countVolwes(str){
         const volwes="aeiou";
         let counter = 0;
         str = str.toLowerCase();
         for (let i = 0; i < str.length; i +=1){

        Сокращенный от одного блока if - else

        let msg = (age >= 10 && age <= 25) ? "Вы допущены" : "Вы не допущены";

        оператор continue 
        Переводит цикл на следующуи итерацию без выполнения кода в цикле 
        for (let i = 1; i<=10; i +=1) {
            if (i === 5){
                continue;
            }
            console.log(i);
        }
        выведет 1234 678910

        оператор break - прерывает цик и полностью выходит из него 
        or (let i = 1; i<=10; i +=1) {
            if (i === 5){
                break;
            }
            console.log(i);
        }
        выведет 1234

        цикл while(condition){}
         испольлзуется в тех условиях когда не извесно колличество итераций
         например при требовании от пользователя ввести пароль цикл if или for только один раз проверит 
         введенный пароль на соответствие и выдаст результат, после чего в случае если пароль не
         верный то для следующей попытки необходимо еще раз запускать if. Но в случае с while цикл
         (вопрос "введите пароль") будет запускаться до тех пор пока не будет введен правильный пароль
         let attempts = 1;
        while (PASSWORD !== userPassword){
            userPassword = prompt("wrong password. Try again");
            attempts +=1;
            if (attempts === 3){
                break;
            }
        }
            if (userPassword === PASSWORD){
                alert("Welcome!");
            }

         цикл do while 
         цикл с постусловием , т.е в отличии от цикла while в do while сначала выпллняется лдействие
         для захода на следующий круг проверяется условие 
         do {} while (condition)

        let i = 0;
        do  {
            console.log(i)
            i +=1;
        }while (i < 10);
        Этот цикл выполняется минимум 1 первый раз. Используется КРАЙНЕ редко
