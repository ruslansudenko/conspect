HTML - HyperText Markup Language язык гипертекстовой разметки
        css - cascading style sheets каскадные таблицы стилей
        source code - исходный код 
        statement - оператор, заявление, утверждение,инструкция (a = b * 2;)
        expression - выражение [ [a] = [ [b] * [2] ] ]  a или 2 или a*2 или полностью 
        undefined - неопределенный
        Parse - анализировать, разбор, структурный анализ, граматический разбор
        initialization - инициализация;
        condition - состояние, условие, положение, обстановка 
        iteration - итерация ( один проход цикла)
