Создание репозитория

В верхнем правом углу своего репозитория на GitHub нажать на "+" и выбрать "new repository". После чего появится новое окно в котором необходимо будет ввести:

1. Repository template.
Для создания нового пустого репо оставляем данный селектор в положении "no template"
Если необходимо создать новый репозиторий на основании другого (в стартовое наполнение скопировать весь код из другого репо ) то в выпадающем списке можно выбрать копируемый репо. НО для этого репозиторий который копируетя в новый должен быть отмечен у себя в settings как Template repository
2. Repository name.
Заполняем имя репозитория. На английском , с маленькой буквы , через дефис
3. Устанавливаем галочку на "Add a README file"
4. Add .gitignore В выпадающем списке выбираем "Node"
5. Нажать "Create repository"
6. Репо создан, создаем ссылку на живую страницу для этого необходимо из репозиторя:
- зайти в settings, выбрать Pages, в разделе "branch" из выпадающего списка "None" выбираем "main" и нажимаем "save"
- подождать нескоторое время и обновить страницу , должно появиться сообщение в рамке "Your site is live at " с сылкой на живую страницу
- скопировать адрес ссылки, вернуться на главную страницу репо, справа возле About нажать на шестеренку, в появившемся окне вставить скопированную ссылку в поле "Website", нажать "save changes" 7. Переносим созданный репо на комп для этого в открывшемся окне нажать на синюю кнопку "< > Code"
В появившемся окне выбрать "Open with GitHub Desktop" после чего откроется приложение GitHub Desktop окном в котором в верхней строчке будет указан путь к репозиторию в "облаке", а в нижней - локальный путь к папке репозитория на компьютере, который, при необходимости, можно изменить.
Нажать "clone" - это создаст локальную версию на пк с привязкой к облаку
8. Нажать "Open in Visual Studio Code" - откроет проект в VS
